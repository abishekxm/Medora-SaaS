# 🏥 Medora – AI Powered Healthcare & Medical Education Platform

> A full-stack AI SaaS platform built with Django, React, PostgreSQL, and AI technologies to simplify healthcare services and support medical students.

---

## 🚀 Live Demo

🌐 Live: https://medora-saas-flax.vercel.app/

⚙️ Backend API: https://medora-saas.onrender.com

---

## 📌 Overview

Medora is a modern healthcare and medical education platform that combines telemedicine with AI-powered learning tools.

Users can:

- 👨‍⚕️ Find doctors
- 📅 Book appointments
- 🎥 Join video consultations
- 💳 Pay consultation fees securely
- 🤖 Chat with an AI assistant
- 🎓 Predict medical colleges
- 📚 Generate AI study materials
- 📝 Take AI-generated quizzes
- 🛣️ Receive AI learning roadmaps

# ⚙️ Installation

## Backend

```bash
git clone <backend_repo_url>

cd backend

pip install -r requirements.txt

python manage.py migrate

python manage.py runserver
```

## Frontend

```bash
cd medora-frontend

npm install

npm run dev
```
